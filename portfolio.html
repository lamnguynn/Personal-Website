All Keys

Swift, UIKit

All Keys is a password manager that allows users to store their credentials securely and utilize them using auto fill. My motivation for making this app was to provide myself and others with a simple and accessible way to store their passwords without having to pay or use the Notes app. I definitely felt this project solidified my knowledge in Swift and built my confidence more as most of the features I had implemented were new and had to be done perfectly to ensure trust from users, and avoid a lawsuit.

Things I learned:

  • Core Data, Keychain, and testing out Realm
  • Using App Store Connect and Publishing
  • Working with entitlements such as App Groups and Keychain Sharing
  • Programmatically creating view controllers and mastering Auto Layout
  • Grand Central Dispatch and Automatic Reference Counting
  • Using UNUserNotificationCenter to add and send notifications
  • Learned Git to upload custom frameworks

Cocoapod Dependencies

Swift, Ruby

From building my apps, I have been able to publish two Cocoapods as a side project: a HalfModalViewController and DropDownButton. My motivation for doing so is that Apple does not provide any options for either to enhance the user experience, so I decided to build my own. Doing so was quite challenging as I had to really break down how each framework would work fundamentally and piece it together. I also learned Git and worked extensively with Unix Terminal to clean and upload these dependencies.

Things I learned:

  • Git commands to upload dependencies
  • Using pod commands to clean and publish dependencies
  • Learning Ruby to make podspec file
  • SOLID Principles and Encapsulation
